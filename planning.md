# Medical Disease Detection Project

## Purpose
Build a machine learning system that predicts:
1. Heart disease risk
2. Liver disease risk
using blood report parameters.

The system combines classical ML models with LLM-based explanation.

## Architecture
- Frontend: React / HTML + JS
- Backend: FastAPI (Python)
- ML Models: Scikit-learn, XGBoost
- LLM: Gemini 3 Flash API
- Database: SQLite / PostgreSQL

## Tech Stack
- Python 3.10
- scikit-learn
- pandas, numpy
- matplotlib, seaborn
- shap (explainability)
- FastAPI
- React
- Gemini 3 Flash API
